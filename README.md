## About

I am a Software Engineering student at **Dalian University of Technology**, exploring how artificial intelligence can be turned into practical and reliable software systems.

My current interests include **AI agents, computer vision, backend architecture and research-oriented intelligent applications**. I prefer learning through complete projects: understanding the problem, designing the system, implementing the core functions and gradually improving the engineering quality.

* Building full-stack AI applications with React, FastAPI and Supabase
* Exploring multi-agent systems and intelligent research workflows
* Learning computer vision and deep learning with PyTorch
* Interested in multimodal intelligence and autonomous systems
* M Award recipient in the Mathematical Contest in Modeling

### [SciPilot](https://github.com/telitor/SciPilot)

A multi-agent research engineering platform designed for software engineering and academic research scenarios.

SciPilot connects five specialized agents into a unified workflow:

* Paper reading and structured analysis
* Complex problem decomposition
* Experimental result interpretation
* Open-source code reproduction
* Research project planning

The platform provides a complete full-stack workflow covering authentication, agent discovery, conversation management, intelligent routing, WebSocket agent communication and persistent data storage.
